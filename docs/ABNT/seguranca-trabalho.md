# ABNT Segurança do Trabalho

Normas brasileiras para segurança dos trabalhadores em ambientes industriais, civis e administrativos.

## Exemplos de Normas

- **NBR 5410:** Instalações elétricas de baixa tensão.
- **NBR 12576:** Sinalização de segurança contra incêndio.
