# ISO 14001 — Gestão Ambiental

Norma para sistemas de gestão ambiental, com foco na sustentabilidade e impacto ambiental.

- Recomendada para empresas de todos os tamanhos.
- Apoia certificações ambientais internacionais.

**Fonte oficial:** [ISO 14001](https://www.iso.org/iso-14001-environmental-management.html)
